MultiNiftiLabelsMasker Class for extracting data from multiple Niimg-like objects using labels of non-overlapping brain regions. MultiNiftiLabelsMasker is useful when data from non-overlapping volumes and from different subjects should be extracted (contrary to :class:`nilearn.maskers.NiftiLabelsMasker`). For more details on the definitions of labels in Nilearn, see the :ref:`region` section.
